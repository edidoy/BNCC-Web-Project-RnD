Untuk setupnya:
- masuk ke folder backend-rnd nya lalu buka terminal
- setelah seperti ini . ./backend-rnd>
- ketik npm install untuk menjalankan server
- setelah sudah selesai, ketik npm start. Jika berhasil akan mengeluarkan output
  "Server nyala di http://localhost:sekian"

API endpoints (dapat menggunakan postman web atau postman)
POST http://localhost:3000/api/feedback
GET http://localhost:3000/api/feedback
GET by ID http://localhost:3000/api/feedback/id
PUT http://localhost:3000/api/feedback/id
DELETE http://localhost:3000/api/feedback/id
