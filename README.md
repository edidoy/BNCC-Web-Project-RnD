Untuk setupnya:
- masuk ke folder backend-rnd nya lalu buka terminal
- setelah seperti ini . ./backend-rnd>
- ketik npm install untuk menjalankan server
- setelah sudah selesai, ketik npm start. Jika berhasil akan mengeluarkan output
  "Server nyala di http://localhost:sekian"
